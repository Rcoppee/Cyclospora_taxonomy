# Assembly

This directory contains the scripts used to map raw Illumina paired-end reads against the *Cyclospora cayetanensis* reference genome and to assess mapping quality.

---

## Files

### `p1_assembling.sh`

This script performs the complete read mapping workflow, including:

- read alignment against the reference genome;
- BAM file sorting and indexing;

### Requirements

Before running the script:

- All paired-end FASTQ files (`*.fastq.gz`) must be located in the same directory.
- The reference genome must also be located in the same directory and named:

```text
ccayetanensis.fasta
```

If another reference genome is used, its filename must be updated accordingly within the script.

### Usage

Run the script from the directory containing both the FASTQ files and the reference genome:

```bash
bash p1_assembling.sh
```

---

### `script_qualimap.sh`

This script performs quality assessment of the mapped BAM files using **Qualimap**.

### Requirements

- Qualimap must be installed before running the script.
- The installation path must be updated in the script (line 16) to match your local installation.
- The script must be executed from the directory containing the BAM files generated by `p1_assembling.sh`.

### Usage

```bash
bash script_qualimap.sh
```

---

## Output

The workflow generates:

- sorted and indexed BAM files;
- mapping statistics;
- Qualimap reports summarizing mapping quality and genome coverage.
